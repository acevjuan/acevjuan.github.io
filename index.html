INSTRUCCIONES

Este programa permite crear, calcular y guardar recetas de cervezas.

  • Las recetas se encuentran guardadas en un array llamado "recetasCreadas". Cada receta es un objeto literal cuyas propiedas describen las características y especificaciones de la cerveza.
  • Para crear, calcular y guardar una nueva receta, hacer click en el botón "Crear receta", la cual solicitará al usuario información necesaria para determinar la cantidad de cebada malteada requerida para hacer la cerveza deseada. Finalmente, creará un objeto dentro del array "recetasGuardadas", gracias a la clase "Recetas".
  • Para el cálculo mencionado en el punto de anterior, se definen como constantes la densidad del agua y la eficiencia de la sala de cocimiento o cervecería.
  • Para modificar alguna receta, hacer click en el boton "Modificar receta". Se requiere indicar el ID de la receta que se desea modificar, la cual aparece en la misma tabla.
  • Para eliminar alguna receta, hacer click en el boton "Eliminar receta". Se requiere indicar el ID de la receta que se desea modificar, la cual aparece en la misma tabla.
  • Para mostrar en la consola todas las recetas, utilizar función "imprimirReceta".

Otras consideraciones a tener en cuenta

  • Hay un archivo independiente para variables, funciones, clases y script.
  • Cada función, variable y clase tiene comentarios en donde se describen su funcionamiento.
  • Para esta entrega no se incluyen restricciones sobre los rangos de valores a digitar para las recetas.
