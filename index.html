Calculador de receta de cervezas

Este programa permite crear, calcular y guardar recetas de cervezas. Actualmente, se debe ejecutar en la consola y funciona de la siguiente manera:

  • Las recetas se encuentran guardadas en un array llamado "recetasCreadas". Cada receta es un objeto literal cuyas propiedas describen las características y especificaciones de la cerveza.
  • Para crear, calcular y guardar una nueva receta, ejecutar la función "crearReceta", la cual solicitará al usuario información necesaria para determinar la cantidad de cebada malteada requerida para hacer la cerveza deseada. Finalmente, creará un objeto dentro del array "recetasGuardadas", gracias a la clase "Recetas".
  • Para el cálculo mencionado en el punto de anterior, se definen como constantes la densidad del agua y la eficiencia de la sala de cocimiento o cervecería.
  • Para modificar alguna receta, utilizar la función "modificarReceta". Se requiere conocer el index del elemento en el array o receta que se desea modificar.
  • Para eliminar alguna receta, utilizar la función "eliminarReceta". Se requiere conocer el index del elemento en el array o recete que se desea eliminar.
  • Para mostrar en la consola todas las recetas, utilizar función "imprimirReceta".

Otras consideraciones a tener en cuenta

  • Hay un archivo independiente para variables, funciones, clases y script.
  • Cada función, variable y clase tiene comentarios en donde se describen su funcionamiento.
  • Para esta entrega no se incluyen restricciones sobre los rangos de valores a digitar para las recetas.
